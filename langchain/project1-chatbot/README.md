# CHATBOT  - TECHNICAL ASSISTANT FOR PROGRAMMERS

Python chatbot using langchain and OpenAI to answer technical questions about programming.

## Technologies
- Python
- Langchain
- OpenAI

# Functionalities
- Technical controlled prompt
- Modular Architecture
- Objective responses

## What does LangChain do?
    Orchestration:
        - Organize the prompt
        - Connection to the LLM
        - Create a predicted execution pipeline
